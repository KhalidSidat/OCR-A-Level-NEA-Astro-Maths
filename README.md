# OCR-A-Level-NEA-Astro-Maths

A browser-based astronomy-inspired timed game built using **HTML, CSS, and JavaScript**.  
This project was created as part of an OCR A-Level NEA (Non-Exam Assessment) to demonstrate my skills in web development and interactive game design.

---------------

About the game

The **Astro Maths** game challenges players to survive a 5-minute session solving maths-related tasks while navigating through an interactive space environment.  
This project helped me learn and apply core web technologies and practice game logic using JavaScript.

Programming Languages used:

- HTML  
- CSS  
- JavaScript

---------------

Demo:

>  Play the game by opening the `index.html` file in a web browser!

---------------

Game Features:

- Countdown timer (5 minutes)
- Incremental timer (Survival Mode) 
- Interactive gameplay  
- Maths-based survival mechanics  
- Styled interface with CSS  
- Multiple HTML game screens for different modes and end displays (start, game, victory, game over)
